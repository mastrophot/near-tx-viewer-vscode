# NEAR Transaction Viewer

Inline transaction insights for VS Code.

## Features

- **Hash Detection:** Recognizes NEAR transaction hashes (base58) in your code.
- **Hover Details:** Shows basic transaction metadata without leaving the editor.
- **Multi-Explorer Support:** Links to both the official NEAR Explorer and Pikespeak.ai.
- **Network Support:** Works with both `mainnet` and `testnet`.

## Usage

Hover over any string that looks like a NEAR transaction hash to see the available explorer links and basic metadata.

## Configuration

- `nearTxViewer.network`: Choose between `mainnet` or `testnet`.
